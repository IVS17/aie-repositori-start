В данном проекте доступны команды:
--report
--overview

К 'report' добавлены 2 параметра:
--title типа str
Этот необязательный параметр добавляет к отчёту в шапке название. По дефолту стоит "# Заголовок"

--show_const типа bool
Этот необязательный параметр позволяет опционально отображать в отчёте дополнительную информацию о наличии константных столбцов. По дефолту отключен
